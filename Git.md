# Инструкция по работе с Git

## Git, что это?

Git - это программа чтобы отследить изменения, произошедшие с проектом, со временем.

### Установка

* Linux

 нужно просто открыть терминал и установить приложение при помощи пакетного менеджера вашего дистрибутива.
 Для Ubuntu команда будет выглядеть следующим образом:

 ```sh
 sudo apt-get install git
 ```

* Windows 

мы рекомендуем git for windows, так как он содержит и клиент с графическим интерфейсом,
и эмулятор bash.
